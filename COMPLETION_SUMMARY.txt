WHAT WAS CHANGED & WHY
================================================================================

CHANGE 1: OpenRouter Headers Support
─────────────────────────────────────
File: backend/app/utils/llm_client.py
Lines: 30-41

BEFORE:
  self.client = OpenAI(
      api_key=self.api_key,
      base_url=self.base_url
  )

AFTER:
  # OpenRouter requires custom headers
  headers = {}
  if "openrouter.io" in (self.base_url or ""):
      headers = {
          "HTTP-Referer": "https://mirofish.ai",
          "X-Title": "MiroFish",
      }
  
  self.client = OpenAI(
      api_key=self.api_key,
      base_url=self.base_url,
      default_headers=headers if headers else None
  )

WHY: OpenRouter requires custom headers for proper request handling,
analytics, and rate limiting. This ensures compatibility while remaining
backward compatible with other providers.

ENVIRONMENT VARIABLES TO FILL IN
================================================================================

REQUIRED (Fill these in .env):
──────────────────────────────

1. LLM_API_KEY
   - What: OpenRouter API key
   - Format: sk-or-v1-xxxxxxxxxxxxxxxx
   - Get from: https://openrouter.ai/keys
   - Cost: FREE (reasonable rate limits)

2. LLM_BASE_URL
   - What: API endpoint
   - Value: https://openrouter.io/api/v1
   - Fixed: Don't change this
   - Cost: N/A

3. LLM_MODEL_NAME
   - What: Model identifier
   - Value: nvidia/nemotron-3-super-120b-a12b:free
   - Quality: Excellent (120B parameters, 128K context)
   - Cost: FREE

4. ZEP_API_KEY
   - What: Zep memory service key
   - Format: zep_xxxxxxxxxxxx
   - Get from: https://app.getzep.com/
   - Cost: FREE (monthly quota)

OPTIONAL (Leave commented if not using acceleration):
──────────────────────────────────────────────────────

5. LLM_BOOST_API_KEY
   - Purpose: Parallel simulation acceleration
   - Format: sk-or-v1-xxxxxxxxxxxxxxxx
   - Get from: https://openrouter.ai/keys (second key)

6. LLM_BOOST_BASE_URL
   - Purpose: Parallel simulation endpoint
   - Value: https://openrouter.io/api/v1

7. LLM_BOOST_MODEL_NAME
   - Purpose: Parallel simulation model
   - Value: nvidia/nemotron-3-super-120b-a12b:free

================================================================================
HOW TO GET API KEYS
================================================================================

STEP 1: OpenRouter API Key (5 minutes)
──────────────────────────────────────
1. Go to: https://openrouter.ai/
2. Click: "Sign Up" or "Sign In"
3. After login, go to: https://openrouter.ai/keys
4. Click: "Create Key" button
5. Copy: The key (starts with sk-or-v1-)
6. Paste: Into LLM_API_KEY in .env

STEP 2: Zep API Key (5 minutes)
───────────────────────────────
1. Go to: https://app.getzep.com/
2. Click: "Sign Up" (create free account)
3. Navigate to: API Keys section (usually in settings/account)
4. Click: "Create Key" or "New Key"
5. Copy: The API key
6. Paste: Into ZEP_API_KEY in .env
