⚡ QUICK START (5 Minutes)

  1. Clone:
     $ git clone https://github.com/666ghj/MiroFish.git
     $ cd MiroFish

  2. Get API Keys:
     • OpenRouter: https://openrouter.ai/keys
     • Zep: https://app.getzep.com/

  3. Configure:
     $ cp .env.example .env
     # Edit .env with your API keys

  4. Install & Run:
     $ npm run setup:all
     $ npm run dev

  5. Open Browser:
     http://localhost:3000
